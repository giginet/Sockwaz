=============
Sockwaz
=============
Sockwaz is realtime tweet viewer written in CoffeeScript.

=============
Instration
=============

require: node, npm

::
  
  git clone git://github.com/giginet/Sockwaz.git
  cd Sockwaz
  sh setup.sh

=============
Settings
=============
You must create settings.json under root of sockwaz project.


See settings.sample.json.
